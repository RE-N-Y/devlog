# Project

1. dev.log is a personal website and research blog.
2. the blog is served as a static website.
3. it's based on astro.js, tailwind css, mdx.
4. tailwind.config.cjs contains global config for tailwind css.
5. astro has a cool feature where you can make components and share it across the app. see src/components as an example.
6. src/content includes content files for publications and mdx file for blogs.
7. the contents are fetched and rendered on src/pages/projects, src/pages/rambling (blog), and src/pages/papers
